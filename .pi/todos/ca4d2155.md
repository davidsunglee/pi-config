## Goal
Automatically commit generated plan and review artifacts after `generate-plan` completes successfully.

## Requirements
- Commit the generated plan file and the review artifact files created during that `generate-plan` run.
- Trigger this only after successful plan generation/review completion.
- Do **not** add push support.
- Do **not** auto-run `execute-plan` after the commit.

## Notes
- Keep plan generation, artifact commit, and plan execution as separate phases.
- The commit should cover only the plan/review artifacts from the completed run, not unrelated working tree changes.
- UX can still confirm with the user before committing, but the feature scope is commit-only (no push, no auto-execute).
